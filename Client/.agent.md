# Client — Agent Workflows

Add route + component (template)

1. Create branch `feat/<ticket>-<feature-name>`.
2. Add component: `Client/src/modules/<Feature>/Feature.vue` and supporting files (composables/services).
3. Register route in `Client/src/router`.
4. Add unit test: `Client/src/__tests__/Feature.spec.ts`.
5. Run `npm ci && npm run test:unit`.
6. Open PR using the PR template.
