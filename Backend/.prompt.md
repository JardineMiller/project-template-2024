# Backend Prompt Templates

Create a MediatR command + handler

```
Create a new class `<Feature>Command : IRequest<<FeatureResponse>>` in ProjectTemplate2024.Application under `<Area>/Commands`.
Add a `<Feature>Handler` implementing `IRequestHandler<<FeatureCommand>, <FeatureResponse>>`.
Add DTOs to ProjectTemplate2024.Contracts and mapping in the Application mapping folder.
```

Controller action stub

```
Add an action in `Controllers/<Area>Controller.cs`:
[HttpPost("<route>")]
public async Task<IActionResult> Create([FromBody] <FeatureRequest> request) => Ok(await _mediator.Send(new <FeatureCommand>(request)));
```
