# Backend — Agent Workflows

Implement API endpoint (template)

1. Create a branch: `feat/<ticket>-<summary>`
2. Add MediatR Request/Command and Handler in `ProjectTemplate2024.Application` following existing folder patterns.
3. Add DTO(s) to `ProjectTemplate2024.Contracts`.
4. Wire mapping in Api.Common.Mapping mapping folder.
5. Add controller action in `Backend/ProjectTemplate2024.Api/Controllers/` to call `IMediator.Send()`.
6. Add unit tests in `ProjectTemplate2024.Tests` (use `TestDbContextFactory` for DB tests or mocks for unit tests).
7. Run `cd Backend && dotnet restore && dotnet build && dotnet test` and fix issues.
8. Open PR with the PR template and checklist.

Add migration (template)

1. Add EF migration in `ProjectTemplate2024.Infrastructure` with `dotnet ef migrations add <Name>`.
2. Document migration and include `dotnet ef database update` instructions if required.
