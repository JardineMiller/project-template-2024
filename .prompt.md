# Agent Prompts & Templates

PR title template

- `<scope>(<area>): short summary [#issue]`

PR description template

- **Summary**: One-paragraph explanation.
- **Changes**: Files added/modified.
- **How to test**: exact commands to run locally.
- **Checklist**: unit tests, integration tests (if applicable), no secrets.

Commit message templates

- `feat(api): add <short description>`
- `test: add <short description>`

Use these templates when the agent opens PRs or generates commit messages.
